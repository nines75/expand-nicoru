# 拡張ニコる

ニコニコ動画でニコるの数に応じてコメントを装飾するUserScript

[ダウンロード](https://github.com/nines75/expand-nicoru/raw/refs/heads/main/src/expand-nicoru.user.js)

## 概要

[nicoExpansion](https://addons.mozilla.org/addon/nicoexpansion/)で提供されていた機能を現在のニコニコ動画で動作するように再現したものです  
オリジナル版の機能に加えていくつかのオプションが存在します

元々は[mico](https://github.com/nines75/mico)の一部として提供していました
